# 🔐 DailyVault

**Your secure, all-in-one offline vault for notes, tasks, links, and documents.**

DailyVault is a privacy-first, completely offline personal organizer that keeps all your important information safely stored on your device. No cloud, no tracking, no internet required.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📝 **Notes Vault** | Jot down ideas, journal entries, and daily thoughts in a clean, distraction-free editor |
| ✅ **Tasks & Reminders** | Track birthdays, bills, medications, and deadlines with categorized reminders |
| 🛒 **Shopping Vault** | Plan groceries and manage shopping lists effortlessly |
| 🔗 **Links Vault** | Bookmark and organize your favorite websites for instant access |
| 📄 **Documents Vault** | Safely store and organize important files |
| 🧠 **Memory Vault** | Save precious memories and personal notes you never want to forget |
| 🔍 **Smart Search** | Quickly find anything across all your vaults |
| 🗑️ **Trash & Recovery** | Accidentally deleted something? Recover it from the trash |
| 🌙 **Dark & Light Mode** | Beautiful, modern UI with theme switching |
| 🔒 **100% Offline** | All data stays on your device — zero cloud, zero tracking |
